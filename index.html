Course Overview

Topics Covered

  • Object-Oriented Programming
  • Classes and Objects
  • Inheritance and Polymorphism
  • Data Structures (Stacks, Queues, Trees)
  • Expression Evaluation
  • Graph Algorithms

Learning Objectives

  • Understand C++ syntax and semantics
  • Implement object-oriented designs
  • Debug and test C++ programs
  • Analyze algorithm complexity

Weekly Content

Week 10: Classes and Objects

Introduction to classes, constructors, destructors, and basic OOP concepts.

Week 15: Advanced Classes

Inheritance, polymorphism, and advanced class features.

Week 16: Data Structures

Stacks, queues, and linked lists implementation with 31 practice questions covering output prediction, error finding, conceptual understanding, and comparative analysis.

Week 17: Expression Evaluation

Infix to postfix conversion and evaluation with 28 practice questions covering conversion exercises, output prediction, and conceptual understanding.

Week 18: Trees and Graphs

Binary trees, graph traversal algorithms.

Week 19: Advanced Algorithms

Sorting, searching, and optimization techniques.

Week 20: File I/O

File operations and data persistence.

Week 21: Templates

Generic programming with templates.

Week 22: STL

Standard Template Library usage.

Week 23: Review

Course review and final projects.
