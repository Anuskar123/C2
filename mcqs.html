Classes & Objects (Questions 1-15)

1. What is the correct syntax to declare a class in C++?

a) class MyClass {};
b) class MyClass;
c) MyClass class {};
d) declare class MyClass;
Answer: a) class MyClass {};

2. Which of the following is NOT a valid access specifier in C++?

a) public
b) private
c) protected
d) internal
Answer: d) internal

3. What is the purpose of a constructor in C++?

a) To destroy objects
b) To initialize objects
c) To copy objects
d) To compare objects
Answer: b) To initialize objects

4. Which operator is used to access members of a class through a pointer?

a) .
b) ->
c) ::
d) *
Answer: b) ->

5. What is the default access specifier for class members in C++?

a) public
b) private
c) protected
d) internal
Answer: b) private

6. Which of the following correctly declares a destructor?

a) ~MyClass() {}
b) MyClass~() {}
c) destroy() {}
d) delete() {}
Answer: a) ~MyClass() {}

7. What is the 'this' pointer in C++?

a) A pointer to the base class
b) A pointer to the current object
c) A pointer to the derived class
d) A pointer to global variables
Answer: b) A pointer to the current object

8. Which keyword is used to define a constant member function?

a) static
b) const
c) final
d) virtual
Answer: b) const

9. What happens when an object is passed by value to a function?

a) The original object is modified
b) A copy of the object is created
c) The object is deleted
d) The function cannot access the object
Answer: b) A copy of the object is created

10. Which of the following is true about static members?

a) They belong to the class, not instances
b) They can only be accessed by const functions
c) They cannot be inherited
d) They must be initialized in the constructor
Answer: a) They belong to the class, not instances

11. What is the purpose of the 'friend' keyword in C++?

a) To declare a function as virtual
b) To allow access to private members
c) To create a copy of an object
d) To define a constant
Answer: b) To allow access to private members

12. Which operator is used for dynamic memory allocation in C++?

a) malloc
b) new
c) alloc
d) create
Answer: b) new

13. What is the correct way to deallocate memory in C++?

a) free()
b) delete
c) destroy()
d) remove()
Answer: b) delete

14. Which of the following is NOT a valid way to initialize an object?

a) MyClass obj;
b) MyClass obj = MyClass();
c) MyClass obj{};
d) MyClass obj = new MyClass();
Answer: d) MyClass obj = new MyClass();

15. What is the difference between struct and class in C++?

a) No difference
b) Struct members are public by default
c) Class members are public by default
d) Struct cannot have functions
Answer: b) Struct members are public by default

Inheritance & Polymorphism (Questions 16-30)

16. Which symbol is used for inheritance in C++?

a) :
b) ::
c) ->
d) .
Answer: a) :

17. What is the default inheritance type in C++?

a) public
b) private
c) protected
d) No default
Answer: b) private

18. Which of the following is true about multiple inheritance?

a) A class can inherit from multiple base classes
b) A class can only inherit from one base class
c) Multiple inheritance is not supported in C++
d) Multiple inheritance requires virtual functions
Answer: a) A class can inherit from multiple base classes

19. What is polymorphism in C++?

a) Ability to take many forms
b) Ability to inherit from multiple classes
c) Ability to have multiple constructors
d) Ability to have private members
Answer: a) Ability to take many forms

20. Which keyword is used to declare a virtual function?

a) abstract
b) virtual
c) override
d) pure
Answer: b) virtual

21. What is a pure virtual function?

a) A function with no implementation
b) A function that cannot be overridden
c) A function that is automatically called
d) A function with default parameters
Answer: a) A function with no implementation

22. Which of the following is true about abstract classes?

a) They cannot be instantiated
b) They must have at least one pure virtual function
c) Both a and b
d) Neither a nor b
Answer: c) Both a and b

23. What is the diamond problem in multiple inheritance?

a) A syntax error
b) Ambiguity in method resolution
c) Memory allocation issue
d) Compilation failure
Answer: b) Ambiguity in method resolution

24. How is the diamond problem solved in C++?

a) Using virtual inheritance
b) Using multiple inheritance
c) Using abstract classes
d) Using templates
Answer: a) Using virtual inheritance

25. Which operator is used for dynamic casting in C++?

a) static_cast
b) dynamic_cast
c) reinterpret_cast
d) const_cast
Answer: b) dynamic_cast

26. What is the purpose of the 'override' keyword?

a) To declare a virtual function
b) To indicate that a function overrides a base class function
c) To prevent overriding
d) To declare a pure virtual function
Answer: b) To indicate that a function overrides a base class function

27. Which of the following is true about protected members?

a) Accessible only within the same class
b) Accessible within the class and derived classes
c) Accessible from anywhere
d) Not accessible in derived classes
Answer: b) Accessible within the class and derived classes

28. What is the base class in the following code?
class Derived : public Base { };

a) Derived
b) Base
c) Both
d) Neither
Answer: b) Base

29. Which inheritance type makes base class public members public in derived class?

a) private
b) protected
c) public
d) None
Answer: c) public

30. What is the output of the following code?
class Base { public: virtual void func() { cout << "Base"; } };
class Derived : public Base { public: void func() { cout << "Derived"; } };
Base* ptr = new Derived(); ptr->func();

a) Base
b) Derived
c) Error
d) Nothing
Answer: b) Derived

Expression Evaluation (Questions 51-65)

51. What is infix notation?

a) Operator between operands
b) Operator after operands
c) Operator before operands
d) No operators
Answer: a) Operator between operands

52. What is postfix notation?

a) Operator between operands
b) Operator after operands
c) Operator before operands
d) Random order
Answer: b) Operator after operands

53. Which algorithm is used to convert infix to postfix?

a) Dijkstra's algorithm
b) Shunting-yard algorithm
c) Kruskal's algorithm
d) Floyd-Warshall algorithm
Answer: b) Shunting-yard algorithm

54. What is the precedence of multiplication over addition?

a) Higher
b) Lower
c) Same
d) None
Answer: a) Higher

55. Which operator has the highest precedence?

a) +
b) *
c) (
d) =
Answer: c) (

56. What is associativity in expressions?

a) Order of evaluation
b) Direction of evaluation
c) Both a and b
d) Neither
Answer: c) Both a and b

57. Which data structure is used in expression evaluation?

a) Queue
b) Stack
c) Linked List
d) Tree
Answer: b) Stack

58. What is the postfix form of "A + B * C"?

a) A B C + *
b) A B C * +
c) A + B * C
d) A B + C *
Answer: b) A B C * +

59. What happens when we encounter an operator in postfix evaluation?

a) Push to stack
b) Pop two operands and apply operation
c) Ignore it
d) Convert to infix
Answer: b) Pop two operands and apply operation

60. Which of the following is left associative?

a) +
b) ^
c) =
d) (unary minus)
Answer: a) +

61. What is the result of evaluating "2 3 4 * +"?

a) 14
b) 20
c) 10
d) 24
Answer: a) 14

62. Which parenthesis has higher precedence?

a) ()
b) []
c) {}
d) All same
Answer: d) All same

63. What is prefix notation?

a) Operator between operands
b) Operator after operands
c) Operator before operands
d) No operators
Answer: c) Operator before operands

64. Which notation is easiest for computers to evaluate?

a) Infix
b) Prefix
c) Postfix
d) All same
Answer: c) Postfix

65. What is the infix form of "A B + C * D /"?

a) ((A + B) * C) / D
b) A + B * C / D
c) A + (B * C) / D
d) (A + B) * (C / D)
Answer: c) A + (B * C) / D

Trees & Graphs (Questions 66-80)

66. What is a binary tree?

a) A tree with at most two children per node
b) A tree with exactly two children per node
c) A tree with no children
d) A tree with unlimited children
Answer: a) A tree with at most two children per node

67. What is the height of a tree?

a) Number of nodes
b) Number of leaves
c) Longest path from root to leaf
d) Shortest path from root to leaf
Answer: c) Longest path from root to leaf

68. Which traversal visits root first?

a) Inorder
b) Preorder
c) Postorder
d) Level order
Answer: b) Preorder

69. Which traversal visits leaves first?

a) Inorder
b) Preorder
c) Postorder
d) Level order
Answer: c) Postorder

70. What is a binary search tree?

a) A sorted binary tree
b) A balanced binary tree
c) A complete binary tree
d) A full binary tree
Answer: a) A sorted binary tree

73. What is a directed graph?

74. Which algorithm finds shortest path in unweighted graph?

75. Which algorithm finds shortest path in weighted graph?
